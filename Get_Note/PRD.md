# 项目说明

本项目是一个笔记同步和处理系统，主要功能是从印象笔记（Evernote/ Yinxiang Biji）的共享笔记本中获取最新笔记，进行处理后同步到飞书表格和印象笔记个人笔记本中。

## 项目结构

- `get_note.py`：主脚本，负责从印象笔记共享笔记本获取最新笔记，处理并保存到本地文件
- `sync_note.py`：同步脚本，负责将处理后的笔记同步到飞书表格和印象笔记个人笔记本
- `config/`：配置文件目录
  - `config.json`：项目配置文件（需要用户创建）
  - `prompt.md`：LLM 摘要提示模板
- `output/`：输出文件目录
  - `note_daily_full.txt`：每日完整笔记内容
  - `note_daily_main.txt`：每日主要笔记内容
  - `notebook_list.json`：笔记本缓存信息
  - `note_processed.json`：处理后的笔记数据
- `demo/`：示例文件目录
- `test/`：测试文件目录

# 支持功能说明

## 1. 笔记获取与处理

### 功能说明
- 从印象笔记共享笔记本中获取最新笔记
- 支持按日期筛选笔记（可配置是否只同步当天笔记）
- 自动提取笔记标题、内容、链接等信息
- 支持下载笔记中的 MP3 附件
- 对笔记内容进行文本处理，提取主要内容和用户留言

### 实现细节
- 使用印象笔记 SDK 连接到共享笔记本
- 通过 API 获取笔记列表和内容
- 对笔记内容进行 ENML 到纯文本的转换
- 支持 MP3 附件的下载和管理
- 维护笔记本同步状态，避免重复处理

## 2. AI 摘要生成

### 功能说明
- 使用 LLM 对笔记内容生成摘要
- 支持多种 LLM 配置（如 qwen、megallm 等）
- 可配置摘要提示模板
- 支持摘要失败的重试机制

### 实现细节
- 通过 OpenAI 兼容的 API 调用 LLM
- 支持不同 LLM 服务的配置和切换
- 对长文本进行截断处理，避免 API 超时
- 记录摘要失败的错误信息

## 3. 钉钉通知

### 功能说明
- 当有新笔记时，通过钉钉机器人发送通知
- 通知内容包含笔记标题、来源和摘要
- 支持自定义通知模板

### 实现细节
- 使用钉钉机器人 webhook 发送文本消息
- 支持消息签名验证
- 对通知失败进行容错处理，不影响主流程

## 4. 飞书同步

### 功能说明
- 将处理后的笔记同步到飞书表格
- 支持批量同步，避免 API 调用限制
- 同步内容包括笔记的标题、内容、摘要、作者等信息

### 实现细节
- 使用飞书开放平台 API
- 获取租户访问令牌进行身份验证
- 批量创建表格记录，每次最多 500 条
- 对 API 错误进行处理和重试

## 5. 印象笔记同步

### 功能说明
- 将处理后的笔记同步到印象笔记个人笔记本
- 支持两种同步模式：
  - 摘要同步：将所有笔记的摘要同步到一个笔记中
  - 全文同步：将指定作者的完整内容同步到一个笔记中
- 支持笔记的创建和更新（追加内容）

### 实现细节
- 使用印象笔记 SDK 连接到个人笔记本
- 自动创建或获取目标笔记本
- 对已有笔记进行内容追加，而非替换
- 支持 HTML 格式的内容渲染

## 6. 笔记本管理

### 功能说明
- 自动检测和管理共享笔记本
- 对长时间未更新的笔记本进行归档处理
- 维护笔记本同步历史，避免重复处理

### 实现细节
- 定期刷新共享笔记本列表
- 对 30 天未更新的笔记本标记为归档
- 维护笔记本的同步状态和历史记录
- 支持从缓存中恢复同步状态
